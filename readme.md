# Foundation Layout Project

![foundation](https://gromcode.s3.eu-central-1.amazonaws.com/front-end/html-css/lesson24/hw1/ouch.png)

### See this site on the [GitHub Pages](https://nimbleninja.github.io/gromcode-foundation-layout/)

Layout supports mobile (up-to-320) and desktop (up-to-1400) versions

### The tech stack is:

- [HTML5](https://ru.wikipedia.org/wiki/HTML)
- [CSS3](https://ru.wikipedia.org/wiki/CSS)
- [Flexbox](https://developer.mozilla.org/ru/docs/Learn/CSS/CSS_layout/Flexbox)
- [Sass (Scss)](https://ru.wikipedia.org/wiki/Sass)
- [BEM methodology](https://ru.wikipedia.org/wiki/%D0%91%D0%AD%D0%9C)
